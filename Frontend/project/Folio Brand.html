01
Die Idee

Das gefaltete Blatt.

Folio bringt Recherche, Quellen und Argument auf ein Blatt — von der ersten Notiz bis zur gebundenen Arbeit.

Herkunft

Das Folio ist der gefaltete Bogen der Buchbinderei — die Grundeinheit jedes gedruckten Werks. Es trägt die Seitenzahl, ordnet das Wissen, hält die Arbeit zusammen. Folio übersetzt diese Jahrhunderte alte Geste der Gelehrsamkeit in einen Arbeitsplatz für das Schweizer Recht.

Das Versprechen

Kein leeres Blatt, das einschüchtert — sondern eines, das mitdenkt: das Quellen erfasst, Belege bereithält und die Argumentationslinie sichtbar macht. Struktur für die langsame, präzise Arbeit, die eine juristische Schrift verlangt.

Das Zeichen

Eselsohr & Marginalie.

06
Bewegung

Die Faltung.

Alles tritt ein, als würde ein Blatt umgeschlagen: Linien schreiben sich, das Eselsohr faltet sich, die Marginalie zieht von oben. Eine Geste, ein Easing — cubic-bezier(.16,1,.3,1), ~600 ms.

Schreiben
Inhalt erscheint zeilenweise, von links — wie geschrieben, nicht eingeblendet.
scaleX 0 → 1 · gestaffelt
Falten
Das Eselsohr klappt aus seiner Ecke ein — die Signaturgeste der Marke.
rotate −58° → 0°
Ziehen
Die Marginalie zieht sich von oben nach unten — der Rand, an dem gedacht wird.
dashoffset → 0

Auf eine Karte klicken zum Abspielen. Bei prefers-reduced-motion erscheint stets der Endzustand.

07
Iconografie

Eine Familie, ein Strich.

24-px-Raster, 1,8-px-Strich, runde Enden. Wo es passt, trägt jedes Icon das Eselsohr oder die indigofarbene Marginalie — so gehören sie sichtbar zur Marke. Fahre über ein Icon.
